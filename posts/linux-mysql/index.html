Linux环境下Mysql安装与部署

Tuesday, Apr 8, 2025 | 2 minute read | Updated at Tuesday, Apr 8, 2025

@ MrZhang

简介

本篇介绍Linux环境下mysql软件的安装步骤！

所需软件

  1. linux的环境可供安装。
  2. mysql5.7安装包
  3. XShell或者MobaXteam远程连接工具

安装步骤

  1. 检查是否已经安装mysql

    rpm -qa | grep mysql

  2. 如果没有安装，什么都不会显示，如果有显示，卸载mysql相关文件，xxx 为文件全名。

    rpm -e xxx

  3. 在官网下载mysql的安装包

    官网地址：https://downloads.mysql.com/archives/community/

  4. 上传解压

    tar -zxvf mysql-5.7.35-linux-glibc2.12-x86_64.tar.gz

  5. 检查mysql的用户组与mysql用户，没有就添加用户

    groups mysql

    添加用户组与用户

    groupadd mysql && useradd -r -g mysql mysql

  6. 创建数据目录并赋予权限

    mkdir -p /data/mysql
chown mysql:mysql -R /data/mysql

  7. 修改配置文件

    如果找不到该配置文件，就创建配置文件

    vim /etc/my.cnf

    修改以下数据

    [mysqld]
bind-address=0.0.0.0
port=3306
user=mysql
basedir=/usr/local/mysql
datadir=/data/mysql
socket=/tmp/mysql.sock
log-error=/data/mysql/mysql.err
pid-file=/data/mysql/mysql.pid
#character config
character_set_server=utf8mb4
symbolic-links=0
explicit_defaults_for_timestamp=true

  8. 初始化

    解压后的mysql-5.7.35-linux-glibc2.12-x86_64文件移动到/usr/local/mysql （文件夹名称修改为mysql不带版本号信息)

    cd /usr/local/mysql/bin/
./mysqld --defaults-file=/etc/my.cnf --basedir=/usr/local/mysql/ --datadir=/data/mysql/ --user=mysql --initialize

  9. 查看初始密码

    cat /data/mysql/mysql.err

  10. 启动mysql

    cp /usr/local/mysql/support-files/mysql.server /etc/init.d/mysql

service mysql start

ps -ef |grep mysql

  11. 登录mysql，使用初始密码

    ./mysql -u root -p   #bin目录下

  12. 修改密码，重新登录

    SET PASSWORD = PASSWORD('root');
ALTER USER 'root'@'localhost' PASSWORD EXPIRE NEVER;
FLUSH PRIVILEGES;

  13. 开放远程访问权限

    use mysql                                            #访问mysql库
update user set host = '%' where user = 'root';      #使root能再任何host访问
FLUSH PRIVILEGES;                                    #刷新

  14. 关闭防火墙可以访问

    systemctl status firewalld  #查看当前防火墙状态
systemctl disable firewalld  #停止防火墙
systemctl stop firewalld #禁用防火墙

    或者可以添加开发的端口号进行访问

    firewall-cmd --state  #防火墙状态
firewall-cmd --zone=public --list-ports  #查看当前开放的端口号

firewall-cmd --zone=public --add-port=80/tcp --permanent   #添加开发80端口
# –permanent永久生效，没有此参数重启后失效
firewall-cmd --zone=public --remove-port=80/tcp --permanent #移除开放的端口

  15. 然后就可以使用本机电脑上的工具去连接mysql数据库了。

  16. 为了使得mysql的命令在所有位置都使用，无需只在bin目录下才可以，可以添加以下配置

    ln -s  /usr/local/mysql/bin/mysql    /usr/bin

  17. 注：mysql低版本默认在linux端是区分大小写的，可以修改配置文件，不区分大小写

    修改不区分大小写,在my.cnf中的[mysqld]后面添加lower_case_table_names=1，重启MYSQL服务

其他操作：=======================================

  1. 修改密码

    1. 开启免密码登陆 修改my.cnf文件   默认在/etc/my.cnf。
vim /etc/my.cnf   在【mysqld】模块下面添加：skip-grant-tables 保存退出。

2. 重启服务，使配置生效 。   
service mysql restart

3. 登陆     
/usr/local/mysql/bin/mysql -u root -p   //不输入密码直接敲回车键

  2. 允许外部访问

    	use mysql 　　　　 #选择访问mysql库
　　update user set host = '%' where user = 'root'; 　　　　 #使root能再任何host访问
　　FLUSH PRIVILEGES; 　　 　　 #刷新

  3. 修改密码

    ALTER USER "root"@"%" IDENTIFIED  BY "root";
FLUSH PRIVILEGES; 　　 　　 #刷新

  4. 退出登录，重新登陆

    把/etc/my.cnf免密删掉。
 在【mysqld】模块下面删除：skip-grant-tables
重启服务    service mysql restart

    登陆 /usr/local/mysql/bin/mysql -u root -p //输入刚修改的密码root敲回车键

  5. 用户权限创建，创建新用户

    创建mysql新用户
CREATE USER 'test'@'%' IDENTIFIED BY '123';
备注上面@后的命令解释

'%' - 所有情况都能访问
‘localhost’ - 本机才能访问
’111.222.33.44‘ - 指定 ip 才能访问
3.给用户授予权限
grant all on 数据库名.数据库表 to 用户名@'%'  identified by "密码";
备注

all 可以替换为 select,delete,update,create,drop
数据库名 所有的 用*
数据库表 所有的 用*
